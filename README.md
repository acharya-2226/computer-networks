# Repository Consisting of Computer Networks Lab Experiments

-This repository contains lab experiments focused on computer networks. Each lab is designed to provide hands-on experience with various networking concepts and tools.
- The experiments cover topics such as Ethernet cable preparation, network device configuration, and the use of essential network commands.

## Lab Experiments Included:
1. [Lab 1: Connection of Network Devices Using Ethernet](Lab%201/Lab1.md)
   - This lab guides you through the process of preparing Ethernet cables with RJ45 connectors, including both straight-through and crossover cables.

2. [Lab 2: Essential Network Commands](Lab%202/lab2.md)
   - This lab introduces various network commands used for diagnosing and troubleshooting network issues, providing syntax and usage for each command.

3. [Lab 3: Using Cisco Packet Tracer](Lab%203/lab3.md)
   - This lab focuses on using Cisco Packet Tracer for simulating network configurations and understanding networking concepts.

4. [Lab 4: Subnetting and Supernetting](Lab%204/lab4.md)
   - This lab covers the concepts of subnetting and supernetting, teaching how to divide and combine IP address spaces effectively.

5. [Lab 5: Configuration of Static Routes and Default Routes](Lab%205/lab5.md)
   - This lab focuses on configuring static routes and default routes in a network, explaining their purposes and how to implement them on network devices.

6. [Lab 6 :Dynamic Routing Configurations : RIP, EIGRP, OSPF, BGP](Lab%206/lab6.md)
   - This lab explores dynamic routing protocols such as RIP, EIGRP, OSPF, and BGP, providing configuration examples and use cases for each protocol.
